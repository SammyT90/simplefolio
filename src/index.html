Projects

Face-Recognition

I created a full stack web application using React for the front end, integrated with an image recognition API. The back end is supported by Node.js and Express, providing a dedicated server to handle user requests, API communication, and MySQL database interactions. Once registered, you can upload pictures and the app will use an API to recognise where the face is on the picture and I have styled the app so that a blue box appears around the face. The app keeps track on how many times a user has uploaded pictures and the user is able to log out and sign in. The MySQL database securely manages user data and images, ensuring data persistence and personalized experiences. The app is hosted on Heroku, guaranteeing global accessibility and responsiveness. This project represents a fusion of technologies, combining React, Node.js, Express, MySQL, and Heroku to deliver an interactive web application with advanced image recognition features.

Quote Generator

The Random Quote Generator is a web app powered by JavaScript, CSS, and HTML. It employs asynchronous JavaScript Fetch requests to retrieve quotes from a REST API. Featuring a straightforward user interface, users can access new quotes with a single click via the 'Generate Quote' button. Additionally, the application includes a Twitter button, enabling users to share their favorite quotes on the platform.
